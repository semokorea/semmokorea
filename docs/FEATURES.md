# 기능 명세서 — 세모코리아 통합 어드민

## 탭 1: 쇼핑몰 운영 어드민

### 1-1. 주문 현황
- 채널별 필터 (전체/네이버/쿠팡/에이블리/지그재그)
- 상태별 필터 (신규/발송대기/발송완료/구매확정)
- 주문 체크박스 다중 선택
- 오늘 지표 카드: 신규주문수, 발송대기, 발송완료, 오늘매출

### 1-2. 배송 처리
- STEP 1: 주문 선택 (주문 현황에서 넘어옴)
- STEP 2: CJ대한통운 운송장 번호 입력 (행별 직접 입력)
- STEP 3: 일괄 발송처리 완료
- STEP 4: 각 플랫폼 배송중 상태 자동 반영

### 1-3. 송장 일괄 출력
- 엑셀 업로드 (사용자 직접 만든 에이블리 57컬럼 양식)
- 컬럼 매핑 (최초 1회 설정 후 자동 저장)
- 주문 데이터 자동 채우기
- 동일 양식 그대로 xlsx 다운로드

## 탭 2: OEM 사업 어드민

### 2-1. 주문 관리
- 거래처별 주문 목록
- 주문번호 형식: OEM-YYMM-NNN
- 상태: 신규접수 → 생산중 → 출고준비 → 출고완료 → 정산완료

### 2-2. 해운 진행상황
- 주문별 6단계 타임라인
  1. 공장 출고 준비중
  2. 중국 물류 대기중
  3. 한국 운송중 (해운)
  4. 통관중
  5. 통관 완료
  6. 고객 출고 완료
- 단계별 날짜, 메모, 예상 도착일 기록
- 수동 업데이트 방식

### 2-3. 견적서 관리
- 견적서 번호 형식: QT-YYMM-NNN
- 품목 최대 10개
- 공급가액 + 부가세(10%) + 배송비 자동 계산
- 상태: 작성중 → 발송완료 → 확정 → 만료
- PDF 출력 (세모코리아 회사 정보 + 직인 포함)

#### 견적서 PDF 구성
```
헤더: 세모코리아 + 견적서 제목 + 견적번호/날짜
수신: 거래처 정보
공급자: 세모코리아 정보 + 직인 이미지
합계금액 배너
품목 테이블 (최대 10개)
소계/부가세/배송비/합계 요약
특이사항
하단: 회사 정보 푸터
```

### 2-4. 거래처 관리
- 거래처 정보: 회사명, 사업자번호, 대표자, 연락처, 이메일, 주소
- 이달 목표 수주액 + 달성률 프로그레스 바
- 거래처별 주문 이력 연결

## 로그인/인증

- 아이디/비밀번호 방식
- 현재: 프론트엔드 하드코딩 (샘플용)
- 실서버: bcrypt + JWT 토큰 방식으로 변경 필요
- 권한: owner(대표) / manager(매니저) / staff(직원)
- 세션 관리: `admin_sessions` 테이블
